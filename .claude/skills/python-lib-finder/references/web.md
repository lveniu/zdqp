# Web Frameworks & APIs

## Web Frameworks (Full Stack)

### Synchronous

| Library | Description | Use Case |
|---------|-------------|----------|
| **django** | Most popular web framework | Enterprise apps, CMS, admin panels |
| **flask** | Microframework | Small to medium apps, quick prototypes |
| **pyramid** | Small, fast, down-to-earth | Flexible architecture needs |
| **masonite** | Modern, developer-centric | Modern web applications |

### Asynchronous

| Library | Description | Use Case |
|---------|-------------|----------|
| **fastapi** | Modern, fast, type hints | REST APIs, auto OpenAPI docs |
| **litestar** | Production-ready ASGI | High-performance async APIs |
| **tornado** | Web framework + async networking | Long-polling, WebSockets |
| **reflex** | Full-stack reactive apps in Python | React-like in pure Python |
| **microdot** | Impossibly small | MicroPython/embedded |
